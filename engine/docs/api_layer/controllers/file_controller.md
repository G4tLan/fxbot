## Purpose
The File Controller handles file downloads, such as logs and exported data from backtests or optimizations.

## Key Endpoints

### `GET /download/{mode}/{file_type}/{session_id}`
Downloads a specific file generated by a session.
- **Parameters**:
    - `mode`: The execution mode (e.g., 'backtest', 'optimize').
    - `file_type`: The type of file (e.g., 'log', 'json', 'csv').
    - `session_id`: The unique ID of the session.
- **Requires Auth**: Yes (via query parameter `token`)
